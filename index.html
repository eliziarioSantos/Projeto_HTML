Como surgiu?

Voltar

A Internet surgiu nos anos 60, na época da Guerra Fria, nos Estados Unidos. O Departamento de Defesa americano pretendia criar uma rede de comunicação de computadores em pontos estratégicos. A intenção era descentralizar informações valiosas de forma que não fossem destruídas por bombardeios se estivessem localizadas em um único servidor."

Em maio de 1995, a rede foi aberta para fins comerciais, ficando a cargo da iniciativa privada a exploração dos serviços. Hoje, para conectar seu computador, o usuário paga os serviços de um provedor de acesso ou tem conexão direta.

HTML

Voltar

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere,este artigo fornecerá uma compreensão básica do HTML e suas funções

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo.

Anatomia de uma Tag

Tags são usados para informar ao navegador a estrutura do site. Ou seja, qaundo se escreve um código em HTML, as tagos seram interpretadas pelonavegador, produzindo assim a estrutura e o conteúdo visual da página.

Primeiro temos a abertura de uma tag, depois o conteúdo e por fim, o fechamento da tag. E isso tudo junto é chamado de elemento.

Elementos podem ter Atributos. eles contèm a infomração exata sobre o elemento.

Exemplos de tags

Voltar

Lembrando que todas as Tags sao colocadas entre <>.

  1. !DOCTPYTE: Define o documento de texto.
  2. script: define o que vai aparecer no lado do cliente.
  3. a: Define o hyperlink.
  4. head: Contém metadados, informações do documento.
  5. header: Define o cabeçalho do documento.
  6. title: define o titulo do documento.
  7. body: Define o corpo do documento.
  8. h1 até h6: Define os titulos.
  9. ul: Define uma lista não ordenada.
  10. ol: Define uma lista ordenada.
  11. li: Define os itens da lista.
  12. br: É como se pulasse uma linha.
  13. small: Define um texto pequeno.
  14. p: Define um paragrafo.
  15. div: Define uma seleção em um documento.
